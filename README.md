# KAF Studio — Блок «Наши услуги»

Реализация UI-экрана на Flutter по предоставленному макету Figma. 
Проект представляет собой интерактивный блок с гексагональной сеткой услуг, плавной анимацией и всплывающими карточками (попапами) при наведении.

## Как запустить проект

1. Убедитесь, что у вас установлен Flutter SDK (версия >= 3.0.0).
2. Склонируйте репозиторий и перейдите в папку проекта:
   ```bash
   git clone https://github.com/BEBEBRR/Flutter_TZ_PC.git
   cd kaf_studio
Подтяните зависимости:

flutter pub get

Запустите проект (рекомендуется запускать в Chrome для оценки десктопного UI):

flutter run -d chrome
📖 Краткое описание и Архитектура
Проект разработан с фокусом на чистоту кода, компонентный подход и переиспользование виджетов согласно ТЗ.
Выполненные требования:
Компонентная архитектура: Интерфейс разбит на независимые, логические StatelessWidget и StatefulWidget (Header, HexGrid, ServiceCard, Popup).
State Management: Управление состоянием наведения (hover) реализовано локально через setState, так как состояние не выходит за рамки изолированных виджетов.
Структура: Строгое разделение на models, screens, widgets и theme.
Адаптивность: Использован FittedBox в связке с MediaQuery. Это позволяет всему холсту пропорционально масштабироваться под любые размеры экрана, сохраняя идеальную пиксельную сетку и не ломая сложную геометрию шестиугольников.
✨ Дополнительно (Бонусные пункты ТЗ):
Оптимизация UI: Максимальное использование const конструкторов для предотвращения лишних ребилдов дерева виджетов.
Анимации: Добавлены плавные микро-интеракции с помощью AnimatedScale, TweenAnimationBuilder и Opacity (плавное увеличение карточек при наведении, плавное появление всплывающих списков).
Fix Web Rendering: Всплывающая карточка обернута в виджет Material для предотвращения багов рендеринга теней в движке Flutter Web.
Пиксель-перфект: Использованы SVG-ассеты для идеального отображения линий на любых разрешениях, а также точная тригонометрическая математика (sin/cos) для расстановки элементов в правильный гексагон.
